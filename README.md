## Documentation

This project is created with an aim to make it easier to understand the concepts of Data Structures and Algorithm in a more better way.

 It includes the following Data Structures and algorithm:
 - Stack
 - Queue
 - Tree
 - Binary Search
 - Selection Sort
 - Insertion Sort
 - Bubble Sort
